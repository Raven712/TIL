# Git hub (2)

### 버전관리와 상관없는 파일 ?

- .gitignore 파일 만들기 ?
  - --> **touch .gitignore 후 내용에 파일명 넣기**
  - 이러면 git status 했을때 untracked files에 , gitignore에 등록된파일은 나오지않음
  - **내용에 *.txt 이런식으로 하면 모든 txt파일이 보이지않게됨 (별표는 "모든"이란 의미 가짐)**
  - **근데 이그노어 하기전에 파일을 커밋후 이그노어파일에 등록하면, 이그노어가 적용이 안됨**

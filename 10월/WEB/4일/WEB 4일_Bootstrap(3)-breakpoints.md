# WEB 4일_Bootstrap(3)-breakpoints..

### breakpoints

- 반응형 제작을 위해 화면크기따라 보이는모습이 달라지는것
  - none : 576픽셀 미만
  - sm: 576~
  - md: 768~
  - lg: 992~
  - xl: 1200~
  - xxl: 1400~



### container

- breakpoint 마다 너비설정을 해주는 기능
  - .container : 각 반응형 중단점에 최대너비 설정
  - .container-sm,md... : 정의된 중단점까지 100퍼센트 너비지정
  - .container-fluid: 모든 중단점에 100퍼설정
